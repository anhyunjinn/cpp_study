<h1>Linked List</h1>
single linked list로 구현
  <h2><조건></h2>
    - 각각의 노드가 1개의 class 일 것<br/>
    - 삽입, 삭제, 검색 기능 구현 
    <h2>linked_list.h</h2>
    연결리스트를 구성하는 노드와 연결리스트의 구조를 정의한 헤더파일
    <h2>linked_list.cpp</h2>
    헤더 파일에서 정의한 함수들을 세부적으로 구현
    <h2>main.cpp</h2>
    linked_list.h와 linked_list.cpp 파일을 실제로 실행하는 파일
